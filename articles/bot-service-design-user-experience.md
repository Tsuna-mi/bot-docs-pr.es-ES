## <a name="text-and-natural-language-understanding"></a><span data-ttu-id="8ecbd-180">Comprensión del lenguaje natural y del texto</span><span class="sxs-lookup"><span data-stu-id="8ecbd-180">Text and natural language understanding</span></span>

<span data-ttu-id="8ecbd-181">Un bot puede aceptar las entradas de **texto** de los usuarios e intentar analizar esas entradas usando la coincidencia de expresiones regulares o las API de **comprensión del lenguaje natural**, como <a href="https://www.luis.ai" target="_blank">LUIS</a>.</span><span class="sxs-lookup"><span data-stu-id="8ecbd-181">A bot can accept **text** input from users and attempt to parse that input using regular expression matching or **natural language understanding** APIs, such as <a href="https://www.luis.ai" target="_blank">LUIS</a>.</span></span> <span data-ttu-id="8ecbd-182">Según el tipo de entrada que proporcione el usuario, la comprensión del lenguaje natural puede ser o no una buena solución.</span><span class="sxs-lookup"><span data-stu-id="8ecbd-182">Depending on the type of input that the user provides, natural language understanding may or may not be a good solution.</span></span>

<span data-ttu-id="8ecbd-183">En algunos casos, un usuario puede estar **respondiendo una pregunta muy específica**.</span><span class="sxs-lookup"><span data-stu-id="8ecbd-183">In some cases, a user may be **answering a very specific question**.</span></span> <span data-ttu-id="8ecbd-184">Por ejemplo, si el bot pregunta "¿Cuál es su nombre?", el usuario puede responder con texto que especifique solo el nombre ("Juan") o con una frase ("Mi nombre es Juan").</span><span class="sxs-lookup"><span data-stu-id="8ecbd-184">For example, if the bot asks, "What is your name?", the user may answer with text that specifies only the name, "John", or with a sentence, "My name is John".</span></span>

<span data-ttu-id="8ecbd-185">Hacer preguntas específicas reduce el ámbito de las posibles respuestas que el bot podría recibir, lo que disminuye la complejidad de la lógica necesaria para analizar y comprender la respuesta.</span><span class="sxs-lookup"><span data-stu-id="8ecbd-185">Asking specific questions reduces the scope of potential responses that the bot might reasonably receive, which decreases the complexity of the logic necessary to parse and understand the response.</span></span> <span data-ttu-id="8ecbd-186">Por ejemplo, considere la siguiente pregunta amplia y abierta: "¿Cómo te sientes?".</span><span class="sxs-lookup"><span data-stu-id="8ecbd-186">For example, consider the following broad, open-ended question: "How are you feeling?".</span></span> <span data-ttu-id="8ecbd-187">Comprender las gran cantidad de permutaciones posibles para responder a ese tipo de pregunta es una tarea muy compleja.</span><span class="sxs-lookup"><span data-stu-id="8ecbd-187">Understanding the many possible permutations of potential answers to such a question is a very complex task.</span></span>

<span data-ttu-id="8ecbd-188">Por el contrario, preguntas específicas como "¿Sientes dolor? Sí/No" y "¿Dónde sientes dolor?
